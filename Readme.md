# Vehicle Data Dashboard

An interactive dashboard for exploring vehicle data using Streamlit.

## File Structure
├── README.md
├── app.py
├── vehicles.csv
└── .streamlit
    └── config.toml

## Installation:
pip install -r requirements.txt
streamlit run app.py

## Libraries Used: 
pandas
scipy
streamlit
altair
plotly
